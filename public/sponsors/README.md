# Sponsor Logos

## Supported formats
- **SVG (preferred)**
- PNG
- WebP

## Naming convention (strict)
- lowercase
- hyphen-separated
- no spaces
- examples: `tectonic.svg`, `chainlink-labs.svg`, `acme-capital.png`

## Dark background guidance
- Prefer white/mono marks for best contrast.
- PNG/WebP must be transparent.

## How to add sponsors
1. Drop logo files into `public/sponsors/`.
2. Add or update entries in `src/data/sponsors.ts` to reference the new file.
